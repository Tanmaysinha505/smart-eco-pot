# smart-eco-pot
This app has real hardware integration (runs on wifi of ESP32 microcontroller) , has the following features : ✅ Live plant monitoring (moisture sensors, pump control) ✅ Functional dashboard with real data ✅ Schedule automation (watering timers) ✅ History tracking (watering logs) ✅ EcoBot UI (chatbot with predefined responses for tips/info)
